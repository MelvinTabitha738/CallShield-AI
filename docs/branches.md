# Branching Policy and Descriptions

### main
Production-ready and stable code. Only merge after testing and review.

### intergration
Integration branch where `frontend`, `backend`, and `ai-model` branches are merged before promoting to main.

### frontend
Handles UI/UX for both web and mobile (React / React Native).

### backend
Handles Django backend logic, APIs, authentication, and database management.

### ai-model
Handles AI/ML model code — speech recognition, NLP, and scam detection intelligence.
